ASPiK SDK
ASPiK Developer's Guide



ASPiK (Audio Specific Plugin Kernel) is the simplest and fastest way to get your professional AU, AAX and VST plugins designed and coded for Windows and MacOS. You only need the target SDKs (all are public/free) and a DAW of your choice. You can design and develop your plugin and assemble the GUI right inside of your DAW while it is running. You can create a new blank plugin project in a few seconds using the ASPiKreator software. And if you are a RackAFX7 user, you can now export your RAFX2 plugins in ASPiK format thanks to an agreement with the author.
